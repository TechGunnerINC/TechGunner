## Project Setup & Contributing Guide

1. **Run These Commands in your Terminal**

  git clone https://github.com/Tech-Gunner/TechGunner.git
  
  cd TechGunner
  
  code .
  
2. **Open Split Terminal in VS code**

3. **Set up Frontend**

  cd Frontend
  
  npm i
  
  npm run format
  
  npm run lint
  
  npm run dev
  
4. **Set up Backend**
   
   cd Backend
   
   npm i
   
   npm run run
