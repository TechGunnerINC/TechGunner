## Project Setup

1. **Clone the project**

```shell
git clone https://github.com/Tech-Gunner/TechGunner.git
cd TechGunner
code .
```
2. **Go to the backend folder and set up the Environment variables**
2. **Run this command in your IDE to initialize and setup the project**

`sh init.sh`
